- 2026-07-21 [repo] Doctrine-graduation candidate surfaced during concurrent-orchestrator-coordination kickoff run 5 (design.md D-13). The principle "correctness over an open-ended partial-failure substrate (a dead process, an unreachable origin, a version-skewed peer) is bounded-or-surfaced, never absolute" was re-derived across FOUR kickoff halts on the same work-division correctness axis: each run asserted an absolute ("always reclaimed / no unit ever dispatched twice") and each found a new interleaving under it. The fix pattern that finally converged (run 5): (1) enumerate the failure-axis matrix once and require every cell answered (D-12), converting "find the next interleaving" (unbounded) into "fill every cell" (finite, checkable); (2) downgrade the top-level guarantee to "best-effort while the substrate is reachable + every residue bounded-and-swept OR durably-surfaced-to-the-operator, never silent" (D-13), which turns spec-vs-mechanism contradictions into verifiable coverage items. Candidate for promotion to a planwright doctrine doc (e.g. a partial-failure-guarantee or coverage-contract rule) that /spec-draft could cite for any spec whose correctness rests on an open-ended partial-failure substrate, so future specs adopt the matrix+downgrade discipline up front rather than after N halts. Seeded here per D-13 rather than promoting unilaterally.
